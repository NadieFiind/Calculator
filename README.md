# Calculator
A terminal-based calculator written in Java.

### !!! VERY UNSTABLE !!!

### Supported Tokens
Whitespaces are ignored.

#### Operators
```
Addition      : +
Subtraction   : -
Multiplication: *
Division      : /
Modulo        : %
Exponentiation: ^
```

#### Brackets
```
Opening Parenthesis: (
Closing Parenthesis: )
```

### Supported Syntax
```yaml
+-
-+
++
--
```

### Unsupported Supported Syntax
```yaml
Decimal Place          : .
Implicit Multiplication: 2(10)
```
