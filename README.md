# Calculator

## This was my second Java program. (im never gonna touch java again)

## !!! VERY UNSTABLE !!!

### Supported Tokens
```yaml
Operators:
  Addition      : +
  Subtraction   : -
  Multiplication: *
  Division      : /
  Modulo        : %
  Exponentiation: ^
Brackets :
  Opening Parenthesis: (
  Closing Parenthesis: )
Digits   :
  0-9
```
### Supported
```yaml
+-
-+
++
--
```
### Not Supported
```yaml
Decimal Place          : .
Implicit Multiplication: 2(10)
```
### Ignored Characters
```yaml
Whitespaces
```
